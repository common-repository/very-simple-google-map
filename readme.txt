== Description ==

This is s very simple plugin for Google map. This plugin will show your location image.You can use this plugin very easily.

== Installation ==

1. Upload `Very simple Google map` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to widget and Place Very simple Google map widget.
4. Enter your Location and specify map's width, height without px.
